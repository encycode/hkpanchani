Project Info

  • Almost every smartphone user is using Whatsapp as primary chat application.


  • So, it makes Whatsapp automatically effective way to market your business or to give any message nowadays.


  • For this purpose we have created this application Bulk-Whatsapp-Sender which allows you to send same customised message to multiple users at same time by just uploading one file.


  • Our tool makes easy to market your business in no time.


It has many features like:
Customised message:

  • You can send customised message with our tool whichever way you like by setting custom name or any other variable.

Automatic contact import:

  • You can import your contacts by just uploading .csv or .txt file from you computer.

Don't need to save contact:

  • You don't have to save contact in which you want to send you message because of file you've uploaded.

Multi account support:

  • You can send same message with multiple Whatsapp account at the same time.

Filter in numbers:

  • Out tool can check for numbers whether that number is having account in whatsapp or not. And can do classification for Whatsapp numbers and non Whatsapp numbers.
